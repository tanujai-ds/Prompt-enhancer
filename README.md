# Prompt Improver

A tool to improve and optimize prompts using AI.

## Structure

- **backend/**: FastAPI backend application
- **frontend/**: Streamlit frontend application

## Setup

1. Install backend dependencies: `pip install -r backend/requirements.txt`
2. Install frontend dependencies: `pip install -r frontend/requirements.txt`
3. Configure `.env` with necessary API keys
4. Run backend: `python backend/main.py`
5. Run frontend: `streamlit run frontend/app.py`
